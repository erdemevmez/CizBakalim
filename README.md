ÇİZ BAKALIM


Gerçek zamanlı çok oyunculu çizim oyunu. 
Her oyuncu ekranın yalnızca bir dilimini çizer 
süre dolunca tüm dilimler birleşir.


Proje Hakkında
Çiz Bakalım, oyuncuların birbirinin çizimini göremeden yalnızca ekranın belirli bir bölümünü çizdiği web tabanlı bir oyundur. Süre sonunda tüm dilimler birleşerek ortaya sürpriz bir resim çıkar.


1)Teknolojiler

-ASP.NET Core MVC (.NET 8)

-SignalR — gerçek zamanlı iletişim

-SQL Server + Entity Framework Core

-HTML5 Canvas + Vanilla JavaScript

-Session tabanlı kimlik doğrulama



2)Özellikler

-Kullanıcı kaydı ve girişi

-Oda oluşturma ve oda koduyla katılma

-Özel oyun süresi (30 / 60 / 90 / 120 saniye)

-Özel tuval boyutu (Küçük / Orta / Büyük)

-Kalem, fırça, silgi, çizgi, dikdörtgen, daire, kova araçları

-Geri al (Ctrl+Z, 40 adım)

-Tekrar izleme — çizim süreci animasyonlu oynatılır

-Tema / konu sistemi

-Sonucu PNG olarak indirme

-Aynı odayla yeniden oynama



3)Kurulum

-Repoyu klonla

-appsettings.json içindeki connection string'i güncelle

-SSMS'de CizBakalim_DB.sql dosyasını çalıştır

-dotnet run komutunu çalıştır

-https://localhost:... adresini aç



4)Planlanan Geliştirmeler

-Profil sayfası ve istatistikler

-Emoji tepkileri

-Karanlık mod

-Mobil uyum iyileştirme

-da şifresi
